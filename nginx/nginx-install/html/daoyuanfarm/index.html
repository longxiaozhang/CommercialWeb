欢迎访问道源农场

高邮市道源农场是一家专门从事传统稻麦种植及深加工、林木、蔬菜种植、水产养殖、畜牧养殖、自主销售、生态旅游为一体的现代农场。
被评定为省级示范农场，总投资近千万。

道远农场推崇健康绿色生态，向社会推出各类产品，主要包括绿色稻米“青亲鲜米”、活水野生鱼类、草鸡、乌骨鸡、野鸭、各类禽蛋以及无公害蔬菜鲜果。
